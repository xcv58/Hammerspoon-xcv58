# Hammerspoon-xcv58

## Installl

1. Download the Hammerspoon from https://www.hammerspoon.org/
2. Clone this repo by: `git clone https://github.com/xcv58/Hammerspoon-xcv58.git`
3. `cd Hammerspoon-xcv58 && zsh ./install.zsh`
4. Open Hammerspoon.app
